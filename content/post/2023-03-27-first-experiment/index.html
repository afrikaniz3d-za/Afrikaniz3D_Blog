I got to learn my modest way around geometry nodes while developing HEXES Mini Pack 04: Electronic Devices. I started work on it in January and was having trouble deciding how many to produce and the style to make them in. Using geonodes made it feel more flexible and user-friendly. The first project was Sensor 21, based on an air quality dataset I explored for a portfolio project.


Made a walkthrough video to show my process for the antenna. Used Kdent Live for the first time and there are things about it I made the switch from Da Vinci Resolve easier - the interface is simpler, and some features are easier to access.

Juggled a lot of things trying to pull this together. Did I achieve what I had set out to do? Yes. Could I do it better if I started over now? Absolutely. Things were spread out over four (4) months to allow for learning some of the tools I needed to use.
