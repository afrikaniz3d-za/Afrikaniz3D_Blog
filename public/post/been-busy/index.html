Been Busy


I got to learn my modest way around geometry nodes while developing HEXES Mini Pack 04: Electronic Devices. I started work on it in January and was having trouble deciding how many to produce and the style to make them in. Using geonodes made it feel more flexible and user-friendly. The first project was Sensor 21, based on an air quality dataset I explored for a portfolio project.

 

I made the above walkthrough video to show my process for the antenna. Used Kdentlive for the first time and there are things about it I made the switch from Da Vinci Resolve easier - the interface is simpler, and some features are easier to access.

Juggled a lot of things trying to pull this together. Did I achieve what I had set out to do? Yes. Could I do it better if I started over now? Absolutely. Things were spread out over four (4) months to allow for learning some of the tools I needed to use.

Started work on the next EDA and I can already tell it will be a better put-together project based on what I’ve learned from the first one. Looking forward to the device design component too.


Taking a break from the weekly pieces was needed, especially when work on HMP04 had a clearer direction. Keto Patrick and the Easter set were done some time ago. Also included some articulated links. Thinking about what I could do with this section in future posts as well.
